# Prestige resets soft-cap the economy curve

**Status:** proposed

## Context

End-game players out-earn the shop. A sink is needed.

## Decision

Prestige resets convert surplus currency into permanent multipliers.

## Consequences

Long-tail retention up; needs careful tuning so resets don't feel punishing.

## Options considered

- **Hard currency cap** — rejected; feels bad, players hoard.

## Cites

- reports/2026-05-18-economy-spike

## Re-open if

The economy spike report shows resets don't move retention.
